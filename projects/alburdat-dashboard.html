Alburdat Dashboard - Sistem Pupuk Presisi IoT

IoT & Mobile Development Flutter | MQTT | ESP32

Deskripsi Proyek

Alburdat Presisi adalah aplikasi dashboard Flutter untuk mengontrol dan memonitor perangkat IoT pemberi pupuk presisi berbasis ESP32. Aplikasi ini terhubung via MQTT untuk kontrol real-time dosis pupuk, monitoring status alat, rekomendasi dosis berdasarkan expert system, dan manajemen statistik. Dengan antarmuka yang responsif, aplikasi dapat berjalan di berbagai platform (Android, iOS, Web, Desktop).

Fitur Utama

  • Monitoring Real-time: Status perangkat (dosis saat ini, total volume, rata-rata, jumlah sesi)
  • Sistem Rekomendasi: Hitung dosis pupuk optimal berdasarkan komoditas tanaman dan umur (HST)
  • Kontrol Manual: Atur dosis pupuk secara manual
  • Statistik Lengkap: Lihat dan reset statistik aplikasi pupuk
  • Manajemen WiFi: Reset koneksi WiFi perangkat untuk konfigurasi jaringan baru
  • Multi-platform: Android, iOS, Web, Desktop (Windows, macOS, Linux)

Teknologi Digunakan

Flutter Dart MQTT ESP32 Provider fl_chart Lottie Google Fonts

Tantangan & Solusi

Komunikasi Real-time dengan ESP32

Menggunakan protokol MQTT dengan broker (Mosquitto/EMQX) untuk memastikan data sensor dan perintah kontrol sampai dengan latency rendah. Implementasi keep-alive dan reconnection logic untuk kestabilan.

Expert System untuk Rekomendasi Dosis

Membangun knowledge base berbasis aturan (rule-based) yang mempertimbangkan jenis tanaman, umur, dan kondisi lingkungan. Dosis dihitung berdasarkan data agronomi dan dapat dikustomisasi.

State Management yang Efisien

Menggunakan Provider untuk mengelola state aplikasi, termasuk status koneksi MQTT, data perangkat, dan rekomendasi, sehingga UI selalu sinkron.

Multi-platform Support

Memanfaatkan kemampuan Flutter untuk membangun aplikasi yang berjalan di Android, iOS, web, dan desktop dengan kode dasar yang sama. Menyesuaikan konfigurasi MQTT untuk setiap platform.

Detail Implementasi

Arsitektur Aplikasi

Aplikasi dibangun dengan arsitektur layered: UI Layer (screens, widgets), Business Logic Layer (providers, services), dan Data Layer (models, MQTT client). Komunikasi dengan perangkat ESP32 dilakukan melalui MQTT broker dengan topik-topik yang telah ditentukan.

Expert System

Sistem rekomendasi dosis pupuk menggunakan rule-based approach dengan data komoditas dan umur tanaman (HST). Basis pengetahuan disimpan dalam format JSON yang dapat diperbarui tanpa perlu rebuild aplikasi.

Grafik Statistik

Menggunakan library fl_chart untuk menampilkan grafik statistik pemupukan, seperti total volume per hari, rata-rata dosis, dan jumlah sesi. Data diambil dari perangkat melalui MQTT dan disimpan sementara di memori.

Keamanan

Menerapkan autentikasi MQTT dengan username/password untuk menghindari akses tidak sah. Untuk reset WiFi, menggunakan mekanisme yang memerlukan konfirmasi pengguna.
