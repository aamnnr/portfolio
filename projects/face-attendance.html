Sistem Presensi Pengenalan Wajah Berbasis IoT

Computer Vision & IoT Juni 2025 - Tugas Akhir

Deskripsi Proyek

Sistem presensi otomatis menggunakan pengenalan wajah dengan algoritma K-Nearest Neighbor (KNN) yang diimplementasikan pada ESP32-CAM. Sistem ini terintegrasi dengan platform web berbasis Flask untuk manajemen data presensi dan user. Digunakan di SMK N 1 Trucuk Klaten sebagai proyek percontohan.

Fitur Utama

  • Presensi dengan pengenalan wajah real-time
  • Verifikasi Senyuman
  • Verifikasi Kedip Mata
  • Dashboard admin untuk manajemen pengguna
  • Ekspor data presensi ke Excel

Teknologi Digunakan

OpenCV ALgoritma KNN Flask Framework gTTS (Google Text-to-Spech MySQL

Tantangan & Solusi

Keterbatasan Memori ESP32

Mengoptimalkan model KNN dan mengurangi dimensi feature vector dengan PCA

Kondisi Pencahayaan Berbeda

Menerapkan preprocessing gambar dengan histogram equalization dan normalisasi
