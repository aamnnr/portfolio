Javis Challenge - Aplikasi Demo Autentikasi

Full Stack Development Oktober 2025 - Technical Challenge

Deskripsi Proyek

Proyek demonstrasi implementasi sistem autentikasi yang aman menggunakan MERN Stack (MySQL, Express.js, React, Node.js) dengan fitur dark mode dan unit testing. Dibangun sebagai solusi technical challenge dengan fokus pada security, scalability, dan code quality.

Aplikasi ini menampilkan best practices dalam pengembangan sistem autentikasi modern dengan proteksi keamanan berlapis dan user experience yang optimal.

Tech Stack

React 19 Vite Node.js Express.js MySQL JWT bcrypt Tailwind CSS Axios Vitest React Testing Library React Router DOM Context API

Arsitektur Aplikasi

Frontend (Client)

  • React 19 - Library JavaScript untuk UI dengan hooks modern
  • Vite - Build tool dan development server yang cepat
  • React Router DOM - Routing untuk Single Page Application
  • Tailwind CSS - Utility-first CSS framework
  • Context API - State management untuk autentikasi dan tema
  • Axios - HTTP client untuk API calls dengan interceptors

Backend (Server)

  • Node.js + Express.js - Runtime dan web framework
  • MySQL2 - Driver MySQL dengan connection pooling
  • JWT - Autentikasi berbasis token dengan expiration
  • bcrypt - Hashing password dengan salt rounds
  • CORS - Cross-Origin Resource Sharing configuration
  • express-rate-limit - Rate limiting untuk keamanan

Tantangan & Solusi

Implementasi HttpOnly Cookies untuk JWT

Mengatasi XSS vulnerability dengan menyimpan token di HttpOnly cookies dan mengimplementasikan proper CORS configuration untuk keamanan cross-origin requests.

State Management untuk Autentikasi

Menggunakan React Context API dengan useReducer untuk mengelola state autentikasi secara global dan memastikan konsistensi data across components.

Testing Async Components

Mengimplementasikan mocking untuk API calls dan context providers menggunakan Vitest dan React Testing Library untuk reliable unit testing.
