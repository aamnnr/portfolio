Sistem Presensi RFID dengan ESP32-CAM

Embedded System Desember 2024 - Kerja Praktik

Deskripsi Proyek

Sistem presensi siswa berbasis RFID yang dikembangkan selama kerja praktik di SMK N 1 Trucuk Klaten. Sistem ini menggabungkan identifikasi kartu RFID dengan capture foto wajah menggunakan ESP32-CAM untuk verifikasi ganda. Data presensi disimpan dalam database lokal dan dapat diakses melalui antarmuka web sederhana.

Fitur Utama

  • Autentikasi dua faktor (RFID + Wajah)
  • Penyimpanan data presensi dengan timestamp
  • Antarmuka web untuk manajemen data siswa dan presensi
  • Ekspor data ke format CSV

Teknologi Digunakan

ESP32 RC522 RFID Kamera PHP MySQL

Tantangan Teknis

Penyimpanan Foto Terbatas

Mengimplementasikan kompresi gambar JPEG dengan kualitas 60% untuk menghemat ruang penyimpanan

Multiple User Access

Membuat sistem antrian data ketika banyak perangkat mengakses database secara bersamaan
